# Guardian Guard

Guardian Guard is a comprehensive child safety application designed to protect children in the digital age. It features a secure backend, real-time monitoring, and a user-friendly interface.

## Features

- **Secure Authentication**: Robust user registration and login system.
- **Real-time Monitoring**: Keep track of activities to ensure safety.
- **Cyber-themed UI**: Modern and engaging interface.
- **Privacy Focused**: Built with privacy and data security in mind.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Express, SQLite
- **Tools**: ESLint, Prettier

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone <YOUR_GIT_URL>
    cd guardian-guard
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
